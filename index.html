This site is here because I really liked the idea behind http://whygitisbetterthanx.com/.
Go there, start using Git, and never ever look back.
It is NOT Linux-only. It is NOT that complicated. It is NOT hard to start with.
It IS ready for Windows developers. It IS ready for YOU!
Try it out. You OWE it to yourself!
This site's idea came from the evil mind of Maciej Aniserowicz who was lately exposed to TFS for the 3rd time in his dev-career. Sadly... years pass, but version control in TFS still sucks.
Maciej was used to working with Mercurial and Git for almost 3 years now, so he vomitted silently in his mouth after several hours with TFS.
Then he started using the uber-excellent Git-TFS.
And so should YOU.
